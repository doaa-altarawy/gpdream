GP-DREAM GIT REPOSITORY
=======================

This repository contains all modules of GP-DREAM.

Each module has a separate directory and needs to be compiled and run independently. For most modules, compiled executables are also included.

See the README files in the corresponding module directories and the GP-DREAM documentation for further information:
https://github.com/marbach/gpdream/wiki/

Join our community: featuring your tool on GP-DREAM is easy and enhances its usability, visibility and impact.
https://github.com/marbach/gpdream/wiki/Contact
